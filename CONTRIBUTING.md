# ¡Bienvenido a la comunidad de Wavv Music!

¡Nos emociona mucho que quieras contribuir al backend de **Wavv Music**! Ya sea que estés solucionando un bug, mejorando la documentación o proponiendo una nueva e increíble funcionalidad, tu ayuda es vital para que este proyecto Open Source siga creciendo.

Colaborar es muy sencillo. Solo sigue estos pasos básicos:

## Pasos para contribuir

### 1. Haz un Fork del repositorio

Haz clic en el botón de **Fork** en la parte superior derecha de esta página para crear una copia del repositorio en tu propia cuenta de GitHub. Luego, clónalo en tu máquina local:

```bash
git clone https://github.com/TU_USUARIO/Backend-Wavv-Music.git
```

### 2. Crea una rama (Branch)

Crea una nueva rama descriptiva para tu cambio. Esto mantiene el orden y facilita la revisión del código:

```bash
git checkout -b feature/nueva-funcion

```

### 3. Realiza tus cambios y haz Commit

Haz los cambios en el código. Asegúrate de que el proyecto siga funcionando correctamente y haz commits con mensajes claros:

```bash
git commit -m "feat: agrega endpoint para obtener canciones recomendadas"
```

### 4. Sube los cambios y abre un Pull Request

Haz push de los cambios a tu repositorio bifurcado (`git push origin feature/nueva-funcion`) y luego dirígete al repositorio original de Wavv Music para abrir un **Pull Request**. Asegúrate de llenar la plantilla de Pull Request describiendo qué cambios realizaste.
> ⚠️ **IMPORTANTE:** Asegúrate de que el Pull Request esté apuntando a la rama **`develop`** (y NO a `main`). El código se probará allí antes de pasar a producción.

¡Y listo! Revisaremos tu aporte lo antes posible. ¡Gracias por ser parte de Wavv Music!
